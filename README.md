# Rozetka Cleaner

Легке Chrome-розширення (Manifest V3), яке прибирає промо-елементи на Rozetka і робить блок ціни більш акуратним.

![Rozetka Cleaner Logo](assets/logo-source.png)

## Що робить

- Ховає червону ціну "Картка Rozetka" у каталозі.
- Ховає блок `rz-red-price` на сторінці товару.
- Ховає банер `rz-product-banner` на сторінці товару.
- Ховає плаваючу кнопку `Rozetka AI`.
- Нормалізує вигляд ціни в картках товару:
  - стара ціна зверху;
  - актуальна ціна знизу;
  - компактний розмір шрифту актуальної ціни.

## Швидкий старт

1. Відкрийте `chrome://extensions`.
2. Увімкніть `Developer mode`.
3. Натисніть `Load unpacked`.
4. Виберіть папку розширення (цей репозиторій).
5. Відкрийте Rozetka і оновіть сторінку.

## Де працює

- Домен: `*.rozetka.com.ua`
- Каталог: приховування червоної "карткової" ціни + `Rozetka AI`
- Сторінка товару: приховування `rz-red-price`, `rz-product-banner`, `Rozetka AI`

## Як це реалізовано

- `styles.css` - основне приховування через CSS-селектори.
- `content.js` - додаткове приховування через JS + `MutationObserver`, щоб елементи не повертались після динамічного рендера.
- `manifest.json` - конфігурація MV3 і підключення контент-скриптів.

## Структура проєкту

- `manifest.json`
- `styles.css`
- `content.js`
- `README.md`

## Кастомізація

Якщо Rozetka змінить DOM/класи:

1. Оновіть селектори в `styles.css`.
2. Оновіть селектори в `content.js` (функції `hidePromoPrices` та `hideRozetkaAIWidget`).
3. Перезавантажте розширення в `chrome://extensions`.

## Важливо

- Проєкт не афілійований з Rozetka.
- Використовуйте на власний розсуд: сайт може змінювати верстку, і селектори потребуватимуть оновлень.
